U17SVv0

shader visual

⚠ Flash warning — some modes flash rapidly. Desktop use recommended.

Controls

  • Click — soft reset. Shift + Click — hard reset.
  • Shift + D — toggle Demo Mode (hard-resets the visual every 10s).
  • Hold a parameter key (below) + ← / → to decrease / increase it; key + 0 stops it changing.
  • Save / Load (top bar) — download & restore the current visual as a .json file (Load turns Demo off).

Parameters

  • a flow speed
  • b horizontal density
  • c detail scale
  • d base color (hue)
  • e lateral waves
  • f ripple / springiness
  • g vertical frequency
  • h background radius
  • i foreground tint
  • j brightness / contrast
  • r rotation
  • t time / phase
  • v glitch amount
  • w warp / mouse offset
  • x horizontal pan
  • y vertical pan
  • z zoom
